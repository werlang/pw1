# Exercício Prático: Painel de Preferências

## Objetivo da Atividade

O objetivo desta prática é ler campos de formulário e refletir essas escolhas em elementos já existentes no DOM.

## Conceitos trabalhados

- `value`;
- `checked`;
- `textContent`;
- `classList`;
- alteração de atributos e classes.

## Especificações Técnicas do Sistema

O aplicativo deve permitir:

- informar nome de usuário;
- escolher uma cor de destaque;
- marcar ou desmarcar uma preferência visual;
- aplicar as mudanças em um card de pré-visualização;
- atualizar textos, classes e estados visuais.

## O que observar durante a prática

- o DOM não serve só para criar elementos; ele também serve para atualizar o que já existe;
- inputs e checkboxes alimentam o estado da interface;
- `classList` ajuda a separar comportamento e aparência.
