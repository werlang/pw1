# Exercício Prático: Galeria de Cards

## Objetivo da Atividade

O objetivo desta prática é criar cards dinamicamente no DOM a partir de valores digitados pelo usuário.

## Conceitos trabalhados

- `createElement()`;
- `append()`;
- `textContent`;
- criação de botões para cada card;
- remoção individual com `remove()`.

## Especificações Técnicas do Sistema

O aplicativo deve permitir:

- informar título e descrição de um card;
- criar um novo card visual a cada cadastro;
- inserir esse card em uma galeria da página;
- remover cards individualmente;
- mostrar quantos cards existem na tela.

## O que observar durante a prática

- cada card precisa ser construído como elemento do DOM;
- botões criados dinamicamente também precisam de comportamento;
- a interface cresce conforme novos elementos são inseridos.
