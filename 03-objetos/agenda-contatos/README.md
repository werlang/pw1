# Exercício Prático: Agenda de Contatos

## Objetivo da Atividade

O objetivo desta prática é cadastrar contatos como objetos e pesquisar dentro de um array de registros estruturados.

## Conceitos trabalhados

- array de objetos;
- `push()`;
- `filter()`;
- `find()`;
- renderização de lista;
- busca por propriedade.

## Especificações Técnicas do Sistema

O aplicativo deve permitir:

- cadastrar nome, telefone e cidade de um contato;
- guardar cada contato em um objeto;
- inserir novos objetos em um array principal;
- listar todos os contatos cadastrados;
- filtrar contatos por nome;
- destacar o primeiro contato encontrado na busca.

## O que observar durante a prática

- quando os dados têm várias propriedades, objetos fazem mais sentido do que arrays simples;
- `filter()` e `find()` funcionam muito bem com listas de objetos;
- a interface passa a depender diretamente das propriedades de cada registro.
